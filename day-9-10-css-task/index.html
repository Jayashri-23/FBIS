This Web Page is created to understand the basics concepts of HTML and CSS.

Inline CSS

1. Inline CSS is written directly inside the HTML tags using the style attributes with property : value; pair.
    Ex.style="color: blue";

Internal CSS

1. This paragraph is styled using internal CSS written inside the head section of the HTML document.
2. Internal CSS is useful when you want to apply styles to a single webpage without using an external stylesheet.

External CSS

External CSS is a method of styling web pages where all CSS rules are written in a separate .css file and then linked to an HTML document.
External CSS is commonly used in the real-world projects.
